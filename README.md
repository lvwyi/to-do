# Todo App — AI 智能待办事项管理

一款现代化的 React + TypeScript 待办事项应用，支持 AI 智能任务拆解、分类管理和本地持久化存储。

## 功能特性

- 📝 创建、编辑、删除待办事项
- 🤖 AI 智能拆解：输入一个复杂任务，自动拆分为可执行的子任务
- 📂 自定义分类（工作/个人/健康/学习/购物）
- 🔍 筛选与排序
- 💾 localStorage 自动持久化
- ⚡ 错误边界保护，防止白屏崩溃

## 快速开始

```bash
npm install
npm run dev
```

## 配置

复制 `.env.example` 到 `.env.local`，填入你的 DashScope API Key：

```bash
cp .env.example .env.local
# 编辑 .env.local，填入你的 API Key
```

获取 API Key：https://dashscope.console.aliyun.com/apiKey
